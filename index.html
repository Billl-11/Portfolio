HealthWise-AI

New Generation AI-Powered Healthcare Conversations

Overview

HealthWise AI is a revolutionary healthcare chatbot powered by a Large Language Model (LLM) integrated with Deep Learning capabilities. The chatbot utilizes Deep Learning to predict diagnoses and aims to provide a human-like conversation experience, allowing users to describe their symptoms as if they were talking to a trusted friend. Its primary goal is to address confusion, frustration, and the search for health information online.

Tech Stack

  • Large Languange Model (LLM - GPT 3.5-turbo): Our chatbot is powered by the LLM GPT 3.5-turbo, allowing it to understand and respond to users fluently in natural language.
  • Deep Learning (ClinicalBert): We employ deep learning, including transfer learning from ClinicalBERT, using symptom-to-disease datasets to facilitate accurate diagnoses.
  • LangChain Framework: The LangChain framework integrates LLM with our self-trained deep learning model. This integration allows LLM to engage in conversations while ensuring precise diagnostic capabilities through deep learning.
  • Cloud Run - GCP: Cloud Run is utilized to deploy the application to the cloud, seamlessly integrated with Github Actions for continuous integration and continuous deployment (CI/CD) in production.

TelkomBot-AI Engine

The Telkom Assistant Bot Wtih GPT-3.5 Large Language Model (LLM)

Overview

TelkomBot-AI Engine is an assistant bot that leverages the large language model GPT-3.5 (Large Language Model) to provide information about Telkom based on data from the telkom.co.id website. This bot will be deployed as a REST API with the ability to store user and bot conversations in a MySQL database.

Tech Stack

  • Large Languange Model (LLM - GPT 3.5-turbo): This chatbot is powered by the LLM GPT 3.5-turbo, allowing it to understand and respond to users fluently in natural language.
  • BeautifulSoup Library: I utilize the BeautifulSoup web scraping library for data extraction and crawling from telkom.co.id.
  • LangChain Framework: The LangChain framework seamlessly integrates LLM with scraped data, allowing LLM to engage in conversations while ensuring precise information sourced from telkom.co.id.
  • Flask Framework: Flask framework is employed for deployment, enabling access to the AI engine through a REST API.
  • MySql Database: The MySQL database is used to store conversations between the user and the bot, allowing users to continue their conversations from where they left off.
  • Cloud Run - GCP: Cloud Run is utilized to deploy the application to the cloud, seamlessly integrated with Github Actions for continuous integration and continuous deployment (CI/CD) in production.

Getting Started

  • Github Repository:
    https://github.com/Billl-11/Telkom-Bot-Engine


TelkomBot Web App

Get to know Telkom Indonesia with TelkomBot

Overview

TelkomBot is a chatbot application that utilizes a Large Language Model (LLM) to provide information to users based on data from the Telkom Indonesia website (telkom.co.id). This application also have databases to keeps a record of user conversations, like ChatGPT.

Tech Stack

  • MySql Database: The MySQL database is used to retrieve past conversations between the user and the bot and then display them on the web interface and also to authorization of login/register.
  • Flask Framework: Flask is employed to facilitate communication between the JavaScript frontend and the AI engine through a REST API.
  • HTML, CSS, Javascript: This tech stack is used to build the frontend of the web app.
  • TelkomBot Rest API: The TelkomBot web app communicates with the bot engine via a REST API that has been deployed. Click here to see how the rest api is created.
  • Cloud Run - GCP: Cloud Run is utilized to deploy the application to the cloud, seamlessly integrated with Github Actions for continuous integration and continuous deployment (CI/CD) in production.
